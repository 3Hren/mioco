Coroutine-based handler library for mio

Using coroutines, an event-based mio model can be simplified to a set of routines, seamlessly scheduled on demand in userspace.

With mioco a coroutines can be used to simplify writing asynchronous io handling in synchronous fashion.

See examples/echo.rs for an example TCP echo server.
